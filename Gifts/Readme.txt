Included gift packs are just demo for Gifts system.

Credits:
- christmas.zip: https://www.iconfinder.com/iconsets/merry-christmas-23
- streamline-emoji.zip: https://www.iconfinder.com/iconsets/streamline-emoji-1

Each gifts archive should be extracted to `content/gifts/%giftPack%` directory. For example:

christmas.zip -> content/gifts/christmas
streamline-emoji.zip -> content/gifts/streamline-emoji
